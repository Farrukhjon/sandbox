Grouping Bits to Bytes and Bytes to Words

1.  Bit = 0/1

2.  Byte = 8 \* Bits

3.  Half-Word = 2 \* Bytes = 16 Bits

4.  Word = 4 \* Byte = 32 Bits

5.  Double-Word = 8 \* Bytes = 64 Bits
