## Category of the built-in data structures

- Sequence structures
    - list
    - tuple
    - str
    - bytes
    - bytearray
- Mapping structures
    - dict
    - SortedDict ("sortedcontainers", an ext lib.)
- Set structures
    - set
    - frozenset
- Queue structures
    - queue.Queue
    - collections.deque
- Other specialized structures
    - array.array
    - heapq
    - collections.namedtuple
    - collections.Counter
