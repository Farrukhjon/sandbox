#The Kotlin programming language
##Main Characteristics of the language:
 - JVM based
 - Statically typed
 - Object-Oriented and Functional paradigms
 - Interoperable with Java and Android
 - Open-Source licensed
 - Concise
 - Null-Safe
 - Tool-friendly
 - CoRoutines
 - Type Aliasing
 
 
Learning Path:
 - Basic/Introduction
 - Advanced
 - Best practice
